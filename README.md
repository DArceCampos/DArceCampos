## Highlighted Projects

- **Semantify Book Reviews** – Kubernetes-based semantic search platform leveraging NLP embeddings with Hugging Face, vector search in Elasticsearch, and hybrid persistence with MariaDB. Implemented a producer–consumer ingestion pipeline using RabbitMQ, caching with Memcached, and a React-based UI supporting vector and full-text search. Fully containerized and automated with Helm on Kubernetes, including end-to-end observability with Prometheus and Grafana.

🔗 https://github.com/carayac/2025-02-IC4302/tree/proyecto-01

- **Products Search V2** – Cloud-native product search engine built on a microservices architecture. Implemented large-scale web scraping with Selenium (500+ products stored in Amazon S3), a Kubernetes ingestion pipeline powered by RabbitMQ, entity extraction with spaCy, and data processing with Spark. Products were indexed in MongoDB Atlas using Atlas Search (facets & highlighting). Exposed through a secure Node.js API on Vercel with a React-based frontend. Fully containerized and automated with Kubernetes and Helm Charts.

🔗 https://github.com/carayac/2025-02-IC4302/tree/proyecto-02

- **Observability & Database Benchmarking** – Kubernetes-based observability and performance evaluation platform comparing SQL and NoSQL engines. Automated deployments were implemented with Helm for Prometheus, Grafana, MariaDB, PostgreSQL, and Elasticsearch. A Flask API instrumented with Prometheus exposed request metrics and cache performance (Redis/Memcached), while load testing with Gatling enabled empirical analysis of latency, throughput, and resource utilization across database engines.

🔗 https://github.com/carayac/2025-02-IC4302/tree/tarea-corta-01

- **Soltura_DB** – Relational sales and inventory management system designed to simulate a real-world business environment. Implemented advanced SQL features including normalized schema design, complex queries optimization, triggers, stored procedures, transactions, and strict referential integrity to ensure data consistency and reliability under concurrent operations.

🔗 https://github.com/AllanDBB/Soltura_DB
